# サイコロを振ってプレゼントが決まる、人生ゲーム的なゲーム

ブラウザゲーム「サイコロを振ってプレゼントが決まる、人生ゲーム的なゲーム」の開発リポジトリです。

現在はPhase 1の最小試作品を進めています。`game`フォルダにはVite＋TypeScriptの開発環境があります。

## 開発サーバーの起動方法

Node.jsとnpmをインストールしたうえで、ターミナルから次のコマンドを順番に実行します。

```bash
cd game
npm install
npm run dev
```

起動後、ターミナルに表示されたURL（通常は `http://localhost:5173/`）をブラウザで開きます。

開発サーバーを終了するときは、起動中のターミナルで `Control + C` を押します。

## 公開用ファイルの確認方法

```bash
cd game
npm run build
```

成功すると、`game/dist`フォルダに公開用ファイルが作られます。
